# pdfpage

## Description

pdfpage is a simple command-line tool to count the pages of pdf files in a
directory.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [To Do](#To-Do)

## Installation

```shell
$ go install github.com/mehdieidi/pdfpage@latest
```

## Usage

```shell
$ pdfpage -d ./cool_pdfs
```
